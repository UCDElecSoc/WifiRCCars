# Controller → ESP32 (Python Client)

## Setup

```bash
cd python
python -m venv .venv
```

### Activate

**Windows**

```bash
.venv\Scripts\activate
```

**Mac/Linux**

```bash
source .venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## Run
Replace IP with the IP of the ESP
```bash
python controller_to_esp.py --ip 192.168.1.123
```
